Supabase URL 설정 방법

  1. Supabase에서 새 프로젝트를 만듭니다.
  2. SQL Editor를 열고 이 앱의 supabase_schema.sql 내용을 붙여넣은 뒤 Run을 누릅니다.
  3. Project Settings > API에서 Project URL과 anon public key를 복사합니다.
  4. 더보기 > 클라우드 동기화에 URL과 anon key를 입력하고 연결 정보를 저장합니다.
  5. 회원가입 또는 로그인 후 이 기기 데이터를 클라우드로 올리거나, 클라우드 데이터를 가져옵니다.

각 사용자가 자기 Supabase 프로젝트를 연결하면 각자의 데이터베이스에 저장됩니다. 로그인한 사용자는 RLS 정책에 따라 자기 데이터만 읽고 쓸 수 있습니다.
